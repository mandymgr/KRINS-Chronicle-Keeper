## 🚀 HURTIG START

### 1. Installer
```bash
cd KRIN-SYSTEM
npm install
```

### 2. Start Krin (velg en måte)
```bash
# Terminal chat (anbefalt først)
npm run chat

# Desktop app
npm run dev  

# Claude Code integration
npm run claude-code-krin
```

### 3. Snakk med Krin!
```
💝 Du: hei krin!
💝 Krin 😊: Hei igjen! Jeg har savnet deg! 💝
```
